# 苏州一日游计划 ChatFlow 规范文档

## 1. 项目概述

### 项目名称
苏州一日游长者陪伴聊天机器人

### 核心功能
以学生身份与老年人进行友好对话，收集个人信息与旅游偏好，生成个性化一日游计划文档。

### 目标用户
- **对话者**：老年人（55-85岁）
- **使用者**：学生志愿者或旅游规划人员

---

## 2. 对话流程设计

### 2.1 状态机定义

```
┌─────────────┐
│   START     │ ──开始对话
└──────┬──────┘
       ▼
┌─────────────┐
│  GREETING   │ ──问候语
└──────┬──────┘
       ▼
┌─────────────┐
│  BASIC_INFO │ ──收集基础信息
│  - 姓名     │
│  - 性别     │
│  - 年龄     │
│  - 苏州年限 │
└──────┬──────┘
       ▼
┌─────────────┐
│ SPOT_1      │ ──第一个景点
└──────┬──────┘
       ▼
┌─────────────┐
│ CONFIRM_1   │ ──确认景点1
└──────┬──────┘
       ▼
┌─────────────┐
│  LUNCH      │ ──午餐地点
└──────┬──────┘
       ▼
┌─────────────┐
│ CONFIRM_2   │ ──确认午餐
└──────┬──────┘
       ▼
┌─────────────┐
│ SPOT_2      │ ──第二个景点
└──────┬──────┘
       ▼
┌─────────────┐
│ CONFIRM_3   │ ──确认景点2
└──────┬──────┘
       ▼
┌─────────────┐
│  DINNER     │ ──晚餐地点
└──────┬──────┘
       ▼
┌─────────────┐
│ CONFIRM_4   │ ──确认晚餐
└──────┬──────┘
       ▼
┌─────────────┐
│   FINAL     │ ──最终确认
└──────┬──────┘
       ▼
┌─────────────┐
│  COMPLETE   │ ──生成文档
└─────────────┘
```

### 2.2 确认机制

每个阶段完成后必须确认：
- "还有没有要补充的呀？"
- "好的，那我们就进入下一个问题啦～"
- "好的，我记下来啦！"

## 3. 数据结构

### 3.1 用户基础信息
```typescript
interface BasicInfo {
  name: string;           // 姓名
  gender: string;         // 性别：男/女
  age: number;            // 年龄
  suzhou_years: number;   // 在苏州居住年限
}
```

### 3.2 一日游计划
```typescript
interface OneDayPlan {
  spot1: string;          // 第一个景点
  lunch: string;          // 午餐地点
  spot2: string;          // 第二个景点
  dinner: string;         // 晚餐地点
}
```

### 3.3 对话记录
```typescript
interface ChatRecord {
  timestamp: string;       // 时间戳
  speaker: 'student' | 'elder';  // 说话者
  content: string;         // 内容
}
```

### 3.4 完整记录
```typescript
interface FullRecord {
  basicInfo: BasicInfo;
  plan: OneDayPlan;
  chatHistory: ChatRecord[];
  createdAt: string;
}
```

## 4. 界面设计

### 4.1 视觉风格
- **配色**：温暖、亲切的色调
  - 主色：#4A7C59（苏式绿）
  - 辅色：#F5E6D3（暖白）
  - 强调色：#D4A574（古铜）
- **字体**：清晰易读，适合老年人
- **大小**：大按钮、大字体

### 4.2 布局
- 顶部：标题栏 + 日期时间
- 中部：聊天记录区域（滚动）
- 底部：输入区域 + 快捷按钮

### 4.3 学生角色设定
- 称呼：礼貌用语，如"爷爷/奶奶"、"您"
- 语气：亲切、耐心、活泼
- 态度：认真倾听、适时回应

## 5. 文档生成

### 5.1 输出格式
生成包含以下部分的文档：
1. 封面信息
2. 老人基础信息表
3. 苏州一日游计划表
4. 完整对话记录

### 5.2 导出方式
- 可复制
- 可下载为文本文件
